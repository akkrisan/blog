Setting Up Rails With HAML

Feb 24th, 2016 5:13 pm

  1. Create a new rails project.

     $ rails new projectName

  2. Add ‘haml’ and ‘haml-rails’ to the Gemfile.

     gem 'haml'
 gem 'haml-rails'

  3. In config/application.rb, add the following code inside the Application class:

     config.generators do |g|
   g.template_engine :haml
 end

  4. Run bundle install.

     $ bundle install
